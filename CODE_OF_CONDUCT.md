# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its
Code of Conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project's issues, pull requests, discussions,
and other community spaces, you agree to abide by its terms.

## Reporting

To report an incident, email the project maintainers or open a private
GitHub Security Advisory on the repository. Reports will be handled
confidentially per the Covenant's enforcement guidelines.

## Scope

This Code of Conduct applies within all project spaces — the repository,
issue tracker, pull request discussions, and any community channels —
and when an individual is representing the project in public spaces.
